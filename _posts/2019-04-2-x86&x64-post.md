> 내 PC -> 속성->시스템->시스템 종류에서 몇비트 운영체제인지 확인 못하는 경우 참고

## Windows 7 또는 Windows Vista

Windows Vista 또는 Windows 7을 사용 중인 경우 32 비트 또는 64 비트 버전 실행 하는지 여부를 확인 하는 두 가지 방법입니다. 작동 하지 않는 경우 다른 해 보세요.

방법 1: 제어판에서 시스템 보기 창

1. **시작** 을 클릭 하 고 검색 상자에 **시스템** 을 입력 한 다음 **제어판** 목록에서 **시스템** 을 클릭 합니다.
2. 운영 체제는 다음과 같이 표시 됩니다.
3. 64 비트 버전 운영 체제: **64 비트 운영 체제****시스템** 에서 **시스템 종류** 에 대 한 나타납니다.
4. 32 비트 버전 운영 체제: **32 비트 운영 체제****시스템** 에서 **시스템 종류** 에 대 한 나타납니다.

방법 2: 시스템 정보 보기 창

1. **시작** 을 클릭 하 고 검색 상자에 **시스템** 을 입력 한 다음 **프로그램** 목록에서 **시스템 정보** 를 클릭 합니다.
2. **시스템 요약** 을 탐색 창에서 선택 하면 운영 체제는 다음과 같이 표시 됩니다.
3. 64 비트 버전 운영 체제: **x64 기반 PC****시스템 종류****항목** 아래 나타납니다.
4. 운영 체제의 32 비트 버전: **x86 기반 PC****시스템 종류****항목** 아래 나타납니다.



## Windows XP Professional

Windows XP 사이트가 있는 경우에 두 가지가 32 비트 또는 64 비트 버전 실행 하는지 여부를 확인 하려면. 작동 하지 않는 경우 다른 해 보세요.

방법 1: 제어판에서 시스템 속성 보기

1. **시작**을 클릭하고 **실행**을 클릭합니다.
2. **Sysdm.cpl** 입력 한 다음 **확인** 을 클릭 합니다.
3. **일반** 탭을 클릭 합니다. 운영 체제는 다음과 같이 표시 됩니다.
4. 64 비트 버전 운영 체제: **Windows XP Professional x64 Edition 버전 < 연도 >****시스템** 아래에 나타납니다.
5. 운영 체제의 32 비트 버전: **Windows XP Professional 버전이 < 연도 >****시스템** 아래에 나타납니다.

방법 2: 시스템 정보 보기 창

1. **시작**을 클릭하고 **실행**을 클릭합니다.
2. **Winmsd.exe** 입력 한 다음 **확인** 을 클릭 합니다.
3. **시스템 요약** 을 탐색 창에서 선택 하면 세부 정보 창에서 **항목** 아래에 있는 **프로세서** 를 찾습니다. 값을 note 합니다.
4. **프로세서** 에 해당 하는 값이 **x86** 로 시작 하는 경우 컴퓨터 32 비트 버전의 Windows 실행 됩니다.
5. **I a 64** 또는 **AMD64****프로세서** 에 해당 하는 값 시작 되 면 컴퓨터 64 비트 버전의 Windows 실행 합니다.
